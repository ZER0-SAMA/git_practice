Git Commands

  • Git add:
            1.git add. :add everything to git for commit
            2.git add file1 file2 file3 : add file to git for commit
            3.git add -p:command to interactively choose which changes to stage
                            y - stage this hunk
                            n - do not stage this hunk
                            q - quit; do not stage this hunk nor any of the remaining ones
                            a - stage this hunk and all later hunks in the file
                            d - do not stage this hunk nor any of the later hunks in the file
                            g - select a hunk to go to
                            / - search for a hunk matching the given regex
                            j - leave this hunk undecided, see next undecided hunk
                            J - leave this hunk undecided, see next hunk
                            k - leave this hunk undecided, see previous undecided hunk
                            K - leave this hunk undecided, see previous hunk
                            s - split the current hunk into smaller hunks
                            e - manually edit the current hunk
                            ? - print help

        
  • Git Branch:
            1.git branch <"branch-new-name"> :creates new branch
            2.git checkout <"branch-new-name"> : switch to new branch
            3.git branch -b <"branch-new-name"> :creates and switch into new branch

    
  • Git stash: 
            1.git stash :store new features in git stash
            2.git stash apply :loads/display latest added git stash
            3.git stash apply index :loads/display specific git stash
            4.git stash push -m "message" :helps in remembering git stash
            5.git stash list :list git stashs
            6.git stash drop index :removes the indexed stash   
            7.git stash pop index : remove from stash and load and ready to commit
            8.git stash clear : clean stash
