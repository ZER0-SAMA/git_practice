Git Commands

  • Git stash: 
            1.git stash
            2.git stash apply
            3.git stash apply index
            4.git stash push -m "message" 
            5.git stash list
            6.git stash drop index :removes the indexed stash   
            7.git stash pop index : remove from stash and load and ready to commit
            8.git stash clear : clean stash
